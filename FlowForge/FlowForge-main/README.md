# FlowForge
 Visual Pipeline | Orchestration Tool | React Flow | FastAPI 

Built a full-stack drag-and-drop pipeline editor enabling users to visually compose and connect modular nodes (Input, LLM, Text, Output, Delete).

Engineered a dynamic DeleteNode with real-time dropdown updates and conditional logic, reducing node management time by 40%.

Eliminated layout breaklines and duplicate renders by refactoring canvas architecture, improving UI responsiveness by 60%.

Designed a registry-based node system for scalable orchestration and clean abstraction, enabling 100% dynamic node injection.

Integrated toast notifications, node palettes, and submission logic for a polished, single-page UX.

Technologies: React • React Flow • FastAPI • Zustand • JavaScript • Python • HTML/CSS • Node Registry Design
